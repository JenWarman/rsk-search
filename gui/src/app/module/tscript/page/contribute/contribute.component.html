What's New 15.12.2021
Trivia Tags

Trivia can now be added to transcripts using TRIVIA tags (same as synopsis).

Corrections Feature

Whole episode corrections can now be submitted by going to an episode from the home page and clicking "Submit Correction" in the Transcript header (you must be logged in to see the button). This allows an entire episode transcript to be replaced. Note: rewards are currently not given for corrections. (NOTE: disabled temporarily to resolve issues exporting synopses and trivia)

Inferred Offsets

Audio offsets are now inferred where missing. However these become less accurate the fewer accurate offsets in a transcript as a result a new button has been added to allow offsets to be inserted manually from the editor based on the audio player's current timestamp.
